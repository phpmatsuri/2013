スポンサー

Diamond Sponsor
CandyCane

CandyCane は、簡単にインストールでき拡張性のあるオープンソースバグ管理システムです。PHP と MySQL を使って作られています。

Platinum Sponsor
株式会社リクルートキャリア

CodeIQ(コードアイキュー)は自分の実力を知りたいITエンジニア向けサービスです。

企業の第一線で働くITエンジニアからの出題に解答し、評価フィードバックを受けることで、自身のスキルがその企業から見て「どの程度のものか」「必要とされるか」を測定できます。

フツーの市場価値じゃない、その企業におけるリアルな評価。気になるあなたは、ぜひトライしてみてください！

Windows Azure

マイクロソフトのクラウドプラットフォーム 「Windows Azure」では、オープンなプラットフォーム提供を強く推進しています。IaaS（Infrastructure as a Service）型のサービスでの Linux サポートを行い、CentOS・Ubuntuなどのディストリビューションのサーバー構築を実現しています。

PHP 開発者の皆様向けには、Windows・Mac・Linux向けのPHP SDKの提供はもちろんの事、PHP ランタイムがビルドインされた PaaS サービス「Windows Azure Web サイト」の提供も開始しております。

Gold Sponsor
株式会社インフィニットループ

「ソースコードでなんでも生み出す」

なんでもない記号から、とんでもないモノを生み出す。

インフィニットループは日々技術を磨き続け、あらゆる難題に答えていく札幌のシステム会社です。
代表作「ブラウザ三国志」をはじめとして、PHPを使った大規模ソーシャルゲームの開発を行っています。

さくらインターネット株式会社

さくらインターネットは、日本におけるインターネットの草創期である1996年から、データセンターサービスを提供しています。自社開発による技術力と自社運営のデータセンターでローコストなサービスを実現。さくらのレンタルサーバやさくらのVPS、さくらのクラウドなど幅広いサービスラインナップを提供し、ITエンジニアを中心に選ばれています。

デルフィネット

ITコンサルタントのプロ、デルフィネットは、日本全国から海外まで、個人経営者から大企業までの会社が抱える経営やオペレーション戦略上の問題に対し、IT技術を駆使してそれらの問題を解決するだけではなく、確実に売り上げを伸ばす為のコンサルティングを行っております。

NTTコミュニケーションズ

Cloudnは、豊富なAPIを備えた拡張性の高いパブリッククラウドサービスです。
PHP開発者の皆様へは、PHPアプリケーションを手軽にスケーラブルに実行可能なCloudn PaaSを提供しています。月額525円から利用可能です。5000円分無料キャンペーン実施中！

Silver Sponsor
株式会社 永和システムマネジメント

私たちはアジャイル開発手法を用いて「お客さまに価値を提供し続けるシステム」を構築し、お客さまと一緒に育てていきます。

過去6年間で100以上のプロジェクトを成功に導いてきました。お客さまのビジネスの決定や変化にシステムを素早く対応させるために、半数のプロジェクトでRubyを使っています。そのほか、Java、PHP、.NETなども積極的に活用しています。

また、2007年からPHPコミュニティへの支援も積極的に行っています。
